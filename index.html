GEA Capital

Pagina hecha para GEA Capital, empresa dedicada al sector inmobiliario con la que trabaje mientras trabajaba en Data Docks, hecha con boostrap y con un formulario de contacto hecho en Laravel. Use JS para crear el efecto deseado por el cliente en el menu.

Trabaje junto al cliente para traer a la vida el diseño que el tenia pensado para su pagina.

Las imagenes fueron adecuadas a la pagina por mi usando el programa de edición de fotos GIMP.

Plasticos Italianos

Landing page y catalogo de una empresa de plasticos, todo se hizo en boostrap, es responsivo, se ve bien en escritorio, tableta y movil. Trabaje junto el cliente para crear la estetica que ellos deseaban, usando las ideas del cliente y haciendo cambios donde se necesitara para dar vida a su visión.

El catalogo fue hecho pensando en ocupar modales para mostrar mas detalles del producto.

Las imagenes que aparecen en la pagina fueron alteradas por mi para que se adecuaran al diseño usando el programa de uso libre GIMP.

Magic Pop

Maquetado y efectos hechos con JS y CSS, con formulario de contacto que guarda la información en una base de datos usando Laravel, el formulario a la vez mandaba correo al administrador de la pagina con la información del cliente y al cliente un correo con una plantilla personalizada hecha en CSS. Se trabajo junto a una diseñadora donde ella me proporcionaba el diseño y las imagenes, mi trabajando siendo pasarlo a codigo y asegurar la fidelidad entre diseño y producto terminado.

Se hizo SEO tecnico en la pagina, asi como el buscador de palabras de Google Ads para determinar las palabras mas relevantes para el ranking de la pagina.

Use GitHub para crear backups y tener el control de las versiones del proyecto, igualmente para trabajar de mejor manera con el becario bajo mi mando

Mine Media Content

Pagina para hecha para una empresa de Radio Difusión, donde se muestran sus objetivos y proyectos. Se uso JS para adecuar ciertas partes del CSS conforme a la maquetación de que me fue proporcionada por nuestra diseñadora. Se creo un reproductor para mostrar los audios del cliente, se uso JS para crear la vista programaticamente y ligar las funciones del reproductor a los botones de la vista (play, pause, volumen, etc).

Se me fue proporcionado una maquetación hecha en Figma desde la que guie para pasar el diseño al codigo, a la vez trabajando de manera conjunta con la diseñadora para resolver cualquier duda y comprobar la viabilidad del diseño

Tienda en Linea

Esqueleto de tienda en linea con todas las funciones relacionadas hecha en Laravel, productos guardados en una base de datos SQL. Se presenta de forma programatica de forma que la vista se actualiza cada vez que se hace un cambio a la base de datos. Con funcion de busqueda basada en nombre y categoria.

La base de datos guarda las ordenes que se hacen en la pagina, asi como los datos del cliente para poder entregar de mejor manera su pedido. Se uso la API de paypal para proporcionar pagos en linea a los clientes.

Reproductor Web

Reproductor Web hecho a medida de los requerimientos del cliente. Se uso JS para mandar a llamar los audios, generar el código HTML y ligar nuestras funciones JS a los botones de la vista generada. Con las funciones que se pueden esperar de un reproductor, play, pause, adelantar, cambiar de audio, etc, así como organizar los audios de más antiguo a más reciente y viceversa.

Dashboard

Dashboard hecho en Laravel al que se accede mediante un login, cuenta con una base de datos SQL en la que se guardan los productos, contactos y ordenes de la tienda, con funciones CRUD (create, read, update y delete) y se puede exportar los registros de la tabla contacto en formato XML.

Life Point Calculator

Proyecto personal hecho en Kotlin, calculadora para el juego de cartas Yugioh, con todo lo necesario para jugar, calculadora, moneda, dado y boton de re start.

Trabajo en Equipo

Tuve la oportunidad de trabajar con un equipo talentoso de tres personas, compuesto por un diseñador y dos programadores. Juntos, creamos sitios web únicos y efectivos que cumplían con las necesidades de nuestros clientes.

Mi papel principal fue coordinar el trabajo entre el diseñador y los programadores, asegurándome de que cada miembro del equipo tuviera las herramientas y recursos necesarios para realizar su trabajo de manera efectiva.

El diseñador comenzó por crear diseños de alta calidad, que se basaron en los requisitos del cliente y los objetivos del proyecto. Con su visión creativa, el diseñador dio vida a los diseños que los programadores convertirían en sitios web.

Resolución de Problemas

Uno de los problemas que enfrentamos en nuestro trabajo fue la falta de contenido para una página web. Para resolver este problema, decidí utilizar una herramienta de inteligencia artificial llamada ChatGPT para acelerar el proceso de creación de contenido. Utilicé esta herramienta para generar texto de alta calidad que cumplía con los objetivos de la página web y las expectativas del cliente. Al utilizar ChatGPT, pude ahorrar tiempo y esfuerzo en la creación de contenido, lo que nos permitió cumplir con los plazos de entrega y entregar resultados satisfactorios al cliente.

Otro problema que enfrentamos fue la identificación de áreas de oportunidad en las páginas web de nuestros clientes. Para resolver este problema, realicé un análisis detallado de cada página web. Una vez identificados los problemas, trabajé con el equipo para implementar soluciones que mejoraran la funcionalidad y la eficiencia de cada página web.
